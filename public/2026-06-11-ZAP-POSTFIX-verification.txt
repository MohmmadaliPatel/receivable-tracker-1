1. src/middleware.ts:
   - Extended to run on /_next/static/:path* (updated matcher).
   - Early check: if a request under /_next/static/ ends with '/', return 404 JSON immediately.

   - Legitimate asset requests never use trailing / on .js/.css chunks.
2. next.config.ts:
   - Added explicit `trailingSlash: false`.
   - Added `productionBrowserSourceMaps: false` (was already default; now documented for client questionnaires).
3. Rebuilt successfully (standalone output valid).
