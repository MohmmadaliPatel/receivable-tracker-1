Taxteck License Generator

Vendor tool — open this file locally in a browser. Generates a signed LICENSE value for the Taxteck Email Auto Manager .env file. Keep LICENSE_SIGNING_SECRET private; it must match on the server that verifies licenses.

Generate once: openssl rand -base64 32

.env example
LICENSE_SIGNING_SECRET=your-32-char-or-longer-secret
LICENSE=paste-generated-token-here
